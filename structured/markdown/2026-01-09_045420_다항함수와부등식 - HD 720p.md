# 요약 정리 — 다항함수(주로 2차·3차·4차)와 부등식 관련 핵심 내용 재구성

간단한 핵심 요약 → 상세 설명(정의·성질·풀이절차·예제) → 응용(패턴, 문제풀이 방법) 순으로 정리. 핵심과 상세를 명확히 분리. 수식은 LaTeX 표기 사용.

---

## 개요 (핵심 메시지)
- 2차 함수 $f(x)=ax^2+bx+c$에서 판별식 $D=b^2-4ac$로 근의 존재·중복 여부 판단.
  - $D>0$ : 서로 다른 실근 $\alpha<\beta$ → 인수분해 $a(x-\alpha)(x-\beta)$.
  - $D=0$ : 중근 $\alpha$ → $a(x-\alpha)^2$.
  - $D<0$ : 실근 없음(허근 쌍).
- 2차 부등식 $ax^2+bx+c>0$ 등은 인수분해 후 부호판(구간검토)으로 해를 결정.
  - $a>0, D>0$이면 해는 $x<\alpha$ 또는 $x>\beta$ 등.
  - $D<0$이면 $a>0$이면 항상 양(절대부등식), $a<0$이면 항상 음.
- 다항식의 근의 **중복도(multiplicity)**에 따라 그래프에서의 동작이 달라짐.
  - 중복도가 홀수이면 근에서 그래프가 '뚫고 지나감'(sign change).
  - 중복도가 짝수이면 근에서 '접촉(touch)' 후 반사(부호 불변).
- 선도계수 $a$의 부호는 전체 종단(끝점) 거동 결정.
  - 최고차항 차수가 짝수이면 양 끝꼴 같음(둘 다 +이면 위로 볼록).
  - 최고차항 차수가 홀수이면 양 끝꼴 반대(좌우 반대 방향).
- 3차 이상 복잡한 경우도 동일한 원리(근의 위치·중복·선도계수)로 해석. 복소근(허근)은 항상 켤레 쌍.
- 그래프적 해석과 대수적(완전제곱·판별식·근의 합·곱) 해석을 연계하면 부등식 풀이와 형태 파악 쉬움.

## 핵심 개념 표(요약)

| 개념 | 정의/식 | 그래프상 의미 |
|---|---:|---|
| 판별식 | $D=b^2-4ac$ | $D>0$: 2개 실근, $D=0$: 중근 1개, $D<0$: 실근 없음 |
| 꼭지점(정점) | $x=-\dfrac{b}{2a},\ y=f\big(-\dfrac{b}{2a}\big)$ | $y$의 최대/최솟값 위치(평행이동으로 유도 가능) |
| 완전제곱 변형 | $ax^2+bx+c = a\big(x+\dfrac{b}{2a}\big)^2 + \Big(c-\dfrac{b^2}{4a}\Big)$ | 대수적·기하적 해석 연결(항상 $\ge$ 또는 $\le$ 형태) |
| 근의 합/곱(2차) | $\alpha+\beta = -\dfrac{b}{a},\ \alpha\beta=\dfrac{c}{a}$ | 축의 방정식 등 도출에 사용 |
| 중복도 영향 | multiplicity $m$가 짝수/홀수 | 짝수: 접촉(부호 불변), 홀수: 통과(부호 전환) |
| 절대부등식 | 식이 모든 실수에 대해 참인 부등식 | ex. $ax^2+bx+c>0$ 모든 실수 참 → 해는 $\mathbb{R}$ |

## 상세 설명 — 교육적 흐름(정의 → 성질 → 예 → 풀이 절차)

### 1) 2차 함수·방정식·부등식

정의
- $f(x)=ax^2+bx+c\ (a\neq0)$

성질
- 판별식 $D=b^2-4ac$로 근의 개수 판단.
- 꼭지점: $x_0=-\dfrac{b}{2a}$, $y_0=c-\dfrac{b^2}{4a}$.
- 축의 방정식: $x=-\dfrac{b}{2a}$ (두 근이 실수든 허근이든 항상 성립).

완전제곱 변형(대수적 유도)
- $f(x)=a\Big(x+\dfrac{b}{2a}\Big)^2 + \Big(c-\dfrac{b^2}{4a}\Big)$.
- $a>0$이면 $f(x)\ge c-\dfrac{b^2}{4a}$, $a<0$이면 $f(x)\le c-\dfrac{b^2}{4a}$.

부등식 풀이(절차)
1. 판별식 계산.
2. 인수분해 가능하면 인수분해, 불가능하면 완전제곱·판별식 논리 사용.
3. 근(실근) 존재 시 근을 기준으로 구간 나누고 부호표(sign chart) 작성.
4. 중복근은 짝수/홀수에 따라 부호 변화 유무 체크.
5. 해는 해당 구간의 합집합으로 표현.

표: 2차 부등식의 일반 해
- 가정: $\alpha<\beta$가 실근, $a>0$
  - $ax^2+bx+c>0 \iff x<\alpha$ or $x>\beta$
  - $ax^2+bx+c<0 \iff \alpha<x<\beta$
  - $ax^2+bx+c\ge0 \iff (-\infty,\alpha]\cup[\beta,\infty)$ (단 중근이면 포함 규정 주의)
- $D<0$인 경우: $a>0$이면 항상 $>0$(해는 $\mathbb{R}$), $a<0$이면 항상 $<0$(해는 공집합)

예제(간단)
- $f(x)=x^2-3x+2=(x-1)(x-2)$
  - $f(x)>0 \iff x<1$ or $x>2$.
- $g(x)=(x-1)^2$
  - $g(x)\ge0$ 항상(절대부등식). $g(x)=0$ 해는 $\{1\}$.

특별한 용어
- 절대부등식(항상 성립하는 부등식): 모든 실수에서 참.
- 공집합 해: 어떤 부등식이 어떤 실수도 만족하지 않을 때.

### 2) 3차 다항식(및 그 이상) — 형태와 근의 중복

정의·기본
- 일반형: $P_3(x)=ax^3+bx^2+cx+d$ ($a\neq0$)
- 인수분해 가능한 경우: $a(x-\alpha)(x-\beta)(x-\gamma)$ 또는 중복근 포함 형태

성질(그래프)
- 선도계수 $a$의 부호가 좌우 끝 동작 결정.
- 근의 중복에 따른 동작:
  - 1차 인수(중복도 1) → 근에서 통과(뚫고 지나감).
  - 2차 인수(중복도 2) → 근에서 접촉(반사).
  - 3차 인수(중복도 3) → 통과하되 국소적으로 3차 형태(꽈배기)처럼 보임.
- 만약 인수 중 하나가 2차(허근을 포함한 실수판별 불가)라면 실근은 하나뿐일 수 있음.

판별·특이점
- 3차식에서 내부 2차인수 $x^2+px+q$의 판별식이 음수이면 그 부분이 허근 쌍 → 실근 1개.
- 다항식의 변곡점, 극값 위치는 미분으로 정확히 찾음(미적분 활용 권장).

예제(형태별)
- $a(x-\alpha)^2(x-\beta)$: $\alpha$에서 접촉, $\beta$에서 통과.
- $a(x-\alpha)(x-\beta)(x-\gamma)$(서로 다른 세 근): 각각 통과, $a>0$이면 좌->우 증가형 말단 등.

부등식 접근
- 동일: 인수분해 → 근과 중복도 확인 → 부호판으로 해 구하기.
- 3차 이상은 구간이 많아지지만 원리는 같음.

주의점
- 복잡한 내부 2차식이 있으면 그 부분의 꼭지점 위치(완전제곱화)나 미분으로 극값을 찾아 전체 모양 확인 필요.

---

### 3) 4차 이상(일반화)

성질 요약
- 최고차 항 차수의 패리티에 따라 양 끝꼴 결정(짝수→같은 방향, 홀수→반대 방향).
- 각 실근의 중복도 규칙(짝수 반사, 홀수 통과) 동일 적용.
- 근이 모두 1차인 경우에는 모든 근에서 통과; 일부 중복이면 해당 위치에서 접촉.
- 복잡성은 단지 근의 수와 중복 조합에 따라 증가.

시각적 원리
- $ax^4$은 $ax^2$와 유사한 기본 형태(단위 확장) — 빠르게 증가.
- 4차의 경우 가능한 패턴(근의 조합)에 따라 많음. 그러나 해석 규칙은 동일.

---

### 4) 복소근(허근) 관련

기본
- 복소근은 항상 켤레 쌍: 만약 $c_1+c_2 i$가 근이면 $c_1-c_2 i$도 근.
- 두 근의 합·곱(실수계수 다항식)은 실수.
  - 예: 허근 $\alpha=c_1+c_2 i,\ \beta=c_1-c_2 i$이면 $\alpha+\beta=2c_1$(실수), $\alpha\beta=c_1^2+c_2^2\ge0$.

응용
- $D<0$인 2차에서 $\alpha\beta>0$(허근이면 곱은 양수 또는 0) — 부등식 판별에 영향.
- 복소근이 있어도 축의 방정식 $x=-\dfrac{b}{2a}$ 등 식은 동일하게 유효(대수적 관계에 근거).

### 5) 부등식 문제풀이 템플릿(일반적 절차)

1. 식 정리(모든 항 한쪽으로 몰기).
2. 다항식 인수분해 시도(근, 중복 확인). 인수분해 안된다면 판별식 또는 근의 공식/수치적 근 추정.
3. 근(실근) 기준으로 수직선 분할.
4. 각 구간에서 부호 결정(테스트값 또는 중복도 규칙 이용).
5. 부등호 종류(>, ≥, <, ≤)에 따라 근 포함 여부 판단.
6. 해의 집합 표기(구간 및 합집합 표기).

예시(3단계)
- 문제: $P(x)>0$인 구간?
- 풀이: 인수분해 → 근 $\{\alpha,\beta,\gamma\}$ → 중복도 고려 → sign chart → 결과.

## 자주 반복되는 주제·패턴 및 의의 분석

1. 판별식의 역할
   - 반복: 2차에서는 판별식 하나로 근의 종류·개수·부등식 해의 전반을 결정.
   - 의의: 계산량 낮추고 대수적 해석 가능.

2. 중복도(parity)의 반복성
   - 반복: 짝수차수 인수는 접촉·부호 불변, 홀수는 통과·부호전환.
   - 의의: 부등식 해의 경계 포함 여부 결정, 그래프 행동 직관적 이해 도와줌.

3. 완전제곱 변형과 기하적 해석 연결
   - 반복: 여러 곳에서 완전제곱을 통해 꼭지점과 최솟값/최댓값 논증.
   - 의의: 대수·기하 연계로 직관과 공식의 상호확인 가능.

4. 수직선(sign-chart) 활용의 일관성
   - 반복: 모든 다항 부등식 문제에서 동일한 방법론 적용.
   - 의의: 체계적 문제풀이로 실수 범위에서 정확하게 범위 나눔.

5. 복소근과 실수계수의 관계
   - 반복: 허근은 켤레 쌍, 합과 곱은 실수 → 축의 위치·중심 계산에 영향.
   - 의의: 허근이 있어도 그래프의 '전반적 추세'는 바뀌지 않음(선도계수·중복도 중심).

## 예제별 핵심 포인트(간단 예시)

- 예1: $f(x)=x^2-3x+2=(x-1)(x-2)$
  - 판별식 $D=1>0$, 실근 1,2
  - $f(x)>0 \iff x<1\ \text{or}\ x>2$.

- 예2: $g(x)=(x-1)^2$
  - $D=0$, 중근 1
  - $g(x)\ge0$ 모든 실수, $g(x)=0$ 해는 $\{1\}$.

- 예3: $h(x)=(x-1)^2(x+2)$
  - 근: $x=1$(중복도2), $x=-2$(중복도1)
  - 부호: $x< -2$: -, $-2<x<1$: +, $x>1$: + (1에서 접촉)
  - $h(x)>0 \iff -2<x<1\ \text{or}\ x>1$ (근 포함 여부에 주의)

## 문제풀이 시 자주 하는 실수(주의사항)
- 중복근에서 부등식의 근 포함 여부를 잘못 처리함(짝수 중근은 부호 불변).
- $D<0$일 때 절대부등식 판단을 빠르게 놓침(예: $a>0$이면 항상 양).
- 인수분해 실패 시 완전제곱·판별식을 병행하지 않음.
- 복소근 존재 시에도 축·근의 합 공식은 유효함을 잊음.

## Key Takeaways (행동 가능·요약 포인트)
- 2차 문제는 판별식 $D$와 완전제곱 변형으로 대부분 해결 가능.
- 부등식 풀 때는 인수분해 → 근 확인 → 중복도 체크 → 부호표 작성 순서로 일관되게 접근.
- 근의 중복도(짝수/홀수)는 그래프의 접촉·통과 여부를 결정하므로 해석의 핵심.
- 복소근이 있어도 근의 합·곱 공식은 실수관계 유지 — 축·평균 위치 계산에 사용.
- 3차 이상도 원리는 동일, 다만 내부 2차식의 성질(판별식)을 확인해 실근 개수·그래프 모양 판단.

Insights (추가 연구·심화 지식, 실무적 팁)

1) 중복도와 도함수 관계 (간단 증명)
- 항목: 다항식 $P(x)$에서 근 $\alpha$의 중복도가 $m$이면 $P(x)=(x-\alpha)^m Q(x)$(단 $Q(\alpha)\ne0$).
- 미분: $P'(x)=m(x-\alpha)^{m-1}Q(x)+(x-\alpha)^m Q'(x)$ → $\alpha$는 $P'$의 근으로서 중복도 $m-1$.
- 결과: 중복도가 짝수이면 $P$와 $P'$의 부호 변화/교차 특징 설명 가능. (극값·접촉 논리의 대수적 근거 제공)

2) Rolle의 정리와 다항식 근 분포
- 서로 다른 두 실근 $\alpha<\beta$가 있으면 $P'(\xi)=0$가 $\alpha<\xi<\beta$에 존재.
- 실근 수 대비 극값 수 관계로 그래프의 가능한 모양 제한 가능(특히 3차의 경우 극값2개 이하 등).

3) 수치적 근 찾기·근 추정 팁
- 인수분해 불가 시: Newton 방법, 이분법, 근 대칭 성질(실수부는 $-\dfrac{b}{a}$에 관련) 사용.
- 정수근 정리(Rational Root Theorem)로 가능한 유리근 후보 목록 생성: $\pm$ (약수 of 상수항)/(약수 of 최고차항).

4) 추가 유용한 판별 규칙들(요약)
- Descartes의 부호 규칙: 양수 근의 상한 개수 추정.
- Sturm의 정리: 실근 수 정확 계산(실계수 다항식에서 구간 내 실근 개수).

5) 부등식 풀이의 실무적 팁
- 인수분해 후 각 인수의 중복도를 테이블로 정리(근과 중복도, 각 구간에서의 부호).
- 단순히 테스트값을 쓰기보다 중복도 규칙을 먼저 적용하면 계산량 감소.
- $D<0$인 2차의 경우 무조건 해가 전체 혹은 공집합이므로 빠르게 결론 가능.

6) 예제(심화) — 중복도 혼합
- $P(x)= (x-1)^3(x+2)^1$ (총 4차)
  - 동작: $x=1$에서 중복도 3이므로 통과하되 국부적인 3차 꼬임, $x=-2$에서 통과.
  - 끝동작: 최고차항 양(계수 양)이면 $x\to-\infty$에서 $-\infty$, $x\to\infty$에서 $\infty$.
  - 부등식 $P(x)>0$ 풀 때 근 1,-2와 구간표로 판단.

7) 그래프 해석 보조: 꼭지점·축 위치 빠르게 찾는 공식
- 꼭지점 x좌표: $x=-\dfrac{b}{2a}$ (2차)
- 꼭지점 y값: $y=c-\dfrac{b^2}{4a}$
- 이 공식들은 암기보다 '완전제곱화'로 유도 가능하므로 상황에 따라 유도해 쓰면 안전.

8) 복소근과 부등식 연결(직관)
- 허근 쌍이 존재하면 그래프는 x축과 교차하지 않음(2차의 경우). 따라서 부등식의 해가 전부 실수 전체 혹은 공집합으로 단순화.
- 복소근의 실수부 크기는 그래프의 중심 위치(축)에 영향.

9) 학습·문제 출제 팁
- 학생에게는 다음을 훈련: (1) 인수분해 숙련, (2) 판별식 해석, (3) sign chart 작성.
- 고난도 문제는 내부 2차의 변위(완전제곱)나 미분으로 접근하는 문제 권장.
