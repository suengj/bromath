## 핵심 개념 (정의 · 성질, LaTeX 표기 사용)
- 단위원(단위원): 중심이 원점, 반지름 $1$인 원.
  - 점 $(\cos\theta,\sin\theta)$로 각 $\theta$의 좌표 정의.
- 라디안(호도법): 각 $\theta$를 반지름 $r$인 원에서 호의 길이 $s$와의 관계로 정의: $\theta = \dfrac{s}{r}$. 단위원에서는 $\theta=s$.
- 사인·코사인 정의(단위원 기준):
  - $\sin\theta = y$-좌표, $\cos\theta = x$-좌표.
- 탄젠트:
  - $\tan\theta = \dfrac{\sin\theta}{\cos\theta}$ (단, $\cos\theta\ne0$).
- 코퍼형·코펑션(동치항):
  - $\cos\left(\dfrac{\pi}{2}+\theta\right) = -\sin\theta$  
  - $\sin\left(\dfrac{\pi}{2}+\theta\right) = \cos\theta$
- 항등식:
  - $\sin^2\theta+\cos^2\theta = 1$
  - $\tan^2\theta+1 = \sec^2\theta$
- 주기성:
  - $\sin(\theta+2\pi)=\sin\theta$, $\cos(\theta+2\pi)=\cos\theta$, $\tan(\theta+\pi)=\tan\theta$

## 문제해결 구조 — 2023 수능(가) 5번 문제 (대화 기반 정리)
- 문제(요약, 발언 근거: 참석자 1·2, 46:07–57:45)
  - 조건: $\tan\theta < 0$  
  - 조건: $\cos\left(\dfrac{\pi}{2}+\theta\right)=\dfrac{1}{\sqrt{5}}$ (대화에서 '5분의 루트오'로 표현; 부호는 대화 문맥상 양수)
  - 목표: $\tan\theta$의 값 결정
- 풀이 전략(요약)
  1. 코사인-사인 관계 이용: $\cos\left(\dfrac{\pi}{2}+\theta\right) = -\sin\theta$.
  2. 주어진 값 대입: $-\sin\theta = \dfrac{1}{\sqrt{5}} \Rightarrow \sin\theta = -\dfrac{1}{\sqrt{5}}$.
  3. 항등식 사용: $\cos^2\theta = 1-\sin^2\theta = 1-\dfrac{1}{5} = \dfrac{4}{5}$ $\Rightarrow$ $|\cos\theta| = \dfrac{2}{\sqrt{5}}$.
  4. 사분면·부호 판단: $\sin\theta<0$이므로 $\theta$는 제3사분면 또는 제4사분면. 추가로 $\tan\theta<0$이므로 $\sin$과 $\cos$의 부호가 서로 다름 → $\cos\theta>0,\ \sin\theta<0$ → 제4사분면 선택.
  5. 따라서 $\cos\theta = \dfrac{2}{\sqrt{5}}$.
  6. $\tan\theta = \dfrac{\sin\theta}{\cos\theta} = \dfrac{-1/\sqrt{5}}{2/\sqrt{5}} = -\dfrac{1}{2}$.
- 결론: $\boxed{\tan\theta = -\dfrac{1}{2}}$ (발언 확인: 참석자들이 풀이 후 정답 일치 확인)

  1. 주어진 삼각식이 동치항(예: $\cos(\pi/2+\theta)$)인지 확인 → cofunction 변환 적용.
  2. 식에서 사인/코사인 값 구하면 제곱합으로 나머지 값 결정.
  3. 부호 판정은 사분면·문제 조건(예: $\tan\theta<0$)으로 결정.

Insights
- 단위원 정의의 이점(간결 정리)
  - 단위원에서 각 $\theta$는 다음과 같이 해석된다: 점 $P(\cos\theta,\sin\theta)$는 원 위의 점이며, 원점에서 $P$로 향하는 반직선과 양의 x축이 만드는 각이 $\theta$이다. 이 정의는 다음 장점 제공:
    - 각의 실수 확장(라디안) 자연스러움.
    - 사분면에 따른 사인·코사인 부호 직관화.
    - 합성각·여러 항등식의 기하적 이해 가능.
- cofunction(여집합) 항등식 유도(간단)
  - $\cos\left(\dfrac{\pi}{2}+\theta\right) = \cos\dfrac{\pi}{2}\cos\theta - \sin\dfrac{\pi}{2}\sin\theta = 0\cdot\cos\theta - 1\cdot\sin\theta = -\sin\theta$  
  - 이 증명은 합성각 공식(코사인 합 공식)을 통해 단순히 확인 가능.
- 호도법(라디안)과 실수 대응
  - 라디안의 핵심: 동일한 원에서 호의 길이 $s$와 중심각 $\theta$는 $s=r\theta$. 단위원($r=1$)이면 $\theta = s$.
  - 연속성: 라디안은 각을 실수로 매끄럽게 표현해주어 미적분(특히 극한·미분)의 정의와 직결.
- 사분면 부호 표(간단 암기)
  - 제1사분면: $\sin>0,\ \cos>0,\ \tan>0$  
  - 제2사분면: $\sin>0,\ \cos<0,\ \tan<0$  
  - 제3사분면: $\sin<0,\ \cos<0,\ \tan>0$  
  - 제4사분면: $\sin<0,\ \cos>0,\ \tan<0$
- 실전 팁
  - cofunction 값(예: $\cos(\tfrac{\pi}{2}+\theta)$)을 바로 떠올리기보다, 단위원 그림 한 번 그려 넣으면 부호·크기 결정 오류 감소.
  - 삼각함수 문제 유형별 핵심 암기 리스트(권장):
    - 특수각 값(30°,45°,60°)  
    - $\sin^2\theta+\cos^2\theta=1$  
    - $\sin(\alpha\pm\beta),\ \cos(\alpha\pm\beta)$ 표준공식(유도 가능)  
    - $\tan\theta=\dfrac{\sin\theta}{\cos\theta}$
- 교육 제언(교재·수업 설계)
  - 입문부: 단위원·라디안·사분면·부호(시각적·예시 중심) — 최소 두 페이지로 자세히 다룰 것.
  - 중간부: 합성각·동치항·항등식(유도 실습) — 학생 스스로 증명해보는 활동 포함.
  - 응용부: 삼각방정식·그래프·주기성(문제 해결 전략) — 문제 유형별 체크리스트 제공.
- 추가 예시 문제(연습)
  1. 주어진: $\cos\left(\dfrac{\pi}{2}-\theta\right)=\dfrac{3}{5}$, $\theta$의 부호 조건 없음. $\sin\theta$와 $\cos\theta$ 구하라.  
     - 풀이 요지: $\cos(\frac{\pi}{2}-\theta)=\sin\theta=\dfrac{3}{5}$ → $\cos\theta=\pm \dfrac{4}{5}$; 부호는 조건에 따라 결정.
  2. 주어진: $\tan(\theta+\pi)=\tan\theta$. 주기성 증명과 응용 문제 제시.
