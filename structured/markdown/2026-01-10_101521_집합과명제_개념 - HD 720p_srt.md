## 1. 핵심 개념 (Core concepts)
(간단 정의 → 주요 식/정리 → 사용 조건)

1. 명제와 조건
- 명제: 참/거짓이 확정되는 문장.  
- 조건(가정): 변수에 따라 참·거짓이 달라지는 식(예: “x는 짝수이다”).  
- 표기: 소문자 p,q (조건/명제). 부정: $\lnot p$.

2. 진리집합 (truth set)
- 정의: 조건 $p$를 참으로 만드는 전체집합 내의 원소들의 집합. 표기: $P$.
- 예: 전체집합 $\{1,\dots,10\}$, 조건 “짝수” → $P=\{2,4,6,8,10\}$.

3. 논리 연산과 집합 대응
- 논리 OR($p\lor q$) ↔ 집합 합집합: $P\cup Q$  
- 논리 AND($p\land q$) ↔ 집합 교집합: $P\cap Q$  
- 부정 $\lnot p$ ↔ 여집합: $P^{c}$

4. 드모르간 법칙 (De Morgan)
- $(P\cup Q)^{c}=P^{c}\cap Q^{c}$  
- $(P\cap Q)^{c}=P^{c}\cup Q^{c}$  
- 일반화: 임의의 집합족에 대해 성립.

5. 포함관계와 명제(함의)
- $p\Rightarrow q$ (P이면 Q이다) 참 ⇔ 진리집합 포함: $P\subseteq Q$.
- 대우(contrapositive): $p\Rightarrow q$ 는 $\lnot q\Rightarrow\lnot p$ 와 동치.
- 역(converse): $q\Rightarrow p$ (동치 아님, 별도 판정 필요).
- 필요조건·충분조건:
  - $p$는 $q$의 충분조건: $p\Rightarrow q$.
  - $q$는 $p$의 필요조건: $p\Rightarrow q$.

6. 양화사 (Quantifiers)
- 존재: “어떤(∃)” = 적어도 하나 존재 (existential)  
- 전체: “모든(∀)” = 모든 원소에 성립 (universal)  
- 부정 관계: $\neg(\forall x\, P(x))\equiv \exists x\, \neg P(x)$, $\neg(\exists x\, P(x))\equiv \forall x\,\neg P(x)$.

7. 부등식 기본
- $a>b \iff a-b>0$; 비교는 차(差)의 부호로 판단.
- 덧셈: $a>b \Rightarrow a+c>b+c$ (모든 실수 $c$).
- 곱셈: $a>b \Rightarrow ac>bc$ if $c>0$; 방향 반전 if $c<0$.
- 0인 케이스 주의(특히 나눗셈·역수 관련 주장).

8. 제곱·절댓값 성질 (절대부등식)
- $x^{2}\ge0$ for all $x\in\mathbb{R}$. (완전제곱)
- $|x|=0 \iff x=0$. $x^{2}=0\iff x=0$.
- $x^{2}+y^{2}=0 \iff x=y=0$.
- 삼각부등식: $|a+b|\le|a|+|b|$ (등호 조건: 같은 부호).
- $|a+b|=|a|+|b| \iff ab\ge0$ (보다 정확히는 $a$와 $b$의 부호가 같을 때).

9. 이차식 판별식과 항상 양수/음수 판단
- $f(x)=ax^{2}+bx+c$ (a>0):
  - 판별식 $\Delta=b^{2}-4ac$.
  - $\Delta<0 \Rightarrow$ 실수에서 항상 $>0$ (또는 항상 $<0$ depending on $a$).
- 예: $x^{2}+x+1$ 는 $\Delta=1-4=-3<0 \Rightarrow$ 모든 실수에서 양수.

10. 산술·기하 평균 (AM-GM)
- $ \dfrac{a+b}{2}\ge\sqrt{ab}$ (양수 $a,b$). 등호 조건: $a=b$.

11. 코시-슈warz (Cauchy–Schwarz)
- $(a^{2}+b^{2})(x^{2}+y^{2})\ge (ax+by)^{2}$; 등호 조건: 벡터가 상수배 관계.
- 일반화: n차원 및 내적공간.

12. 포함-여집합·차집합
- 집합 차: $X\setminus Y = X\cap Y^{c}$.
- 부분집합: $C\subseteq B \iff C\setminus B=\varnothing$.
- 집합 등호: $A=B \iff A\subseteq B$ 및 $B\subseteq A$.

13. 부분집합 개수 및 조합
- 집합 $|A|=n$ 이면 부분집합 수 $2^{n}$.
- 원소 수가 $k$인 부분집합 수: $\binom{n}{k}$.
- 순서를 고려한 경우: 순열 $P(n,k)=n(n-1)\cdots(n-k+1)=\dfrac{n!}{(n-k)!}$; 집합(순서 무시)일 때는 나눗셈(중복 고려) 필요(예: $\dfrac{n(n-1)}{2}$).

14. 포함배제법 (Inclusion–Exclusion)
- 두 집합: $|A\cup B|=|A|+|B|-|A\cap B|$.
- 세 집합: $|A\cup B\cup C|=\sum|A|-\sum|A\cap B|+|A\cap B\cap C|$.

## 2. 세부 설명·예시·실전 유의점 (Details & Examples)

표: 개념 → 핵심 식/관찰 → 실전 유의점

| 개념 | 핵심 식/관찰 | 실전 유의점 |
|---|---:|---|
| 논리 부정 | $\lnot(p\lor q)\equiv\lnot p\land\lnot q$ | 문장 부정 시 “또는”→“그리고”로 바뀜; 집합에선 합집합의 여집합 = 각 여집합의 교집합 |
| 역·대우·부정 | 역: $q\Rightarrow p$; 대우: $\lnot q\Rightarrow\lnot p$ (동치) | 대우로 증명하면 종종 더 쉬움. 역은 동치 아님 |
| 곱셈 시 부호 | $a>b$ 에 $c$ 곱할 때 $c>0$이면 방향 유지, $c<0$이면 방향 반전 | $c=0$인 경우 주의. 시험문제에서 함정으로 자주 출제 |
| 나눗셈(분모) | $b\neq0$ 조건 항상 명시 | 분모가 0이면 식 자체가 정의되지 않음 (내신에서 꼼수 문제) |
| 제곱 비교 | 양수 영역에서 $a>b \iff a^{2}>b^{2}$ (단, 부호 고려) | 음수 영역은 순서 뒤바뀜 → 그래프(예: $y=x^{2}$)로 직관 확인 |
| 삼각부등식 | $|a+b|\le|a|+|b|$ | 등호: 같은 방향(부호)일 때 |
| 완전제곱 비음수 | $(a\pm b)^{2}\ge0$ → 전개로 유용한 부등식 도출 | 항등적으로 항상 성립, 등호 조건 파악 중요 |
| AM-GM 활용 | $\dfrac{a+b}{2}\ge\sqrt{ab}$ | 최대/최소 문제에서 우선 검토 (가능하면 AM-GM 시도) |
| C-S 활용 | $(\sum a_{i}^{2})(\sum x_{i}^{2})\ge(\sum a_{i}x_{i})^{2}$ | 벡터 관점(내적)으로 이해하면 확장 쉬움 |
| 이차식 항상 양수 | $\Delta<0$이면 $ax^{2}+bx+c$ 는 $a>0$ 일때 항상 $>0$ | 판별식 계산과 등호 발생 여부(최솟값 위치) 확인 |
| 논리적 포함 | $p\Rightarrow q$ ⇔ $P\subseteq Q$ | 집합 그림(벤)으로 직관화 후 증명(부분집합 포함관계로 전환) |
| 부분집합 수 | $2^{n}$ (총) / $\binom{n}{k}$ (크기 k) | 조건부(특정 원소 포함/미포함) 문제는 선택지 수 곱셈으로 처리 |

몇 가지 구체 예시·주의:
- 분수 부등식: $\dfrac{A}{B}>1 \Rightarrow A>B$ 라고 쓸 때 반드시 $B>0$ 가정 필요. (시험 함정)
- $|x|$ 관련: $|x|=0\iff x=0$, $|x|+|y|=0\iff x=y=0$ — 등호의 필요충분 조건 활용 자주 등장.
- 판별식 예: $x^{2}+x+1$ 은 모든 실수에서 양수 → 부등식에서 하한값 활용.
- 절댓값 제곱을 이용한 증명: 삼각부등식 증명 시 양변 제곱 가능(모두 비음수 조건).

## 3. 문제 해결 구조 (Problem-solving patterns)

1. 명제 증명
- 직접증명 (direct): 가정 → 정리까지 연쇄적으로 전개.
- 대우증명 (contrapositive): $\lnot q\Rightarrow\lnot p$ 증명 → 원명제 참. (효율적)
- 귀류법 (reductio ad absurdum): 결론을 부정하여 모순 도출 → 원명제 참. (루트의 무리수 증명 예시: 루트3 무리수 증명)

2. 부등식(비교) 문제
- 가능한 경우 분류: 부호(case) 분석 (특히 변수의 부호가 식에 영향 줄 때).
- 0 특이점 체크: 곱셈/나눗셈 과정에서 0이 빠지지 않도록 조건 명시.
- 등호(=) 성립 조건 추적: 완전 제곱·AM-GM·C-S 등에서 언제 등호가 되는지(동일 성질 확인).

3. 집합 문제
- 그림(벤다이어그램)으로 먼저 시각화.
- 표현 방식: “A이고 B이고 C는 아니다” 등으로 원소 조건 기술 → 연산식으로 전환($A\cap B\cap C^{c}$).
- 포함·차·여집합 전환으로 간단화.

4. 조합 문제
- 자유 선택(각 원소 포함/미포함): 이진(2) 선택 → $2^{n}$.
- “특정 원소 포함” 조건: 해당 원소 고정 후 나머지에 대한 선택 수 계산.
- ‘순서 무시’ 문제: 순열로 셈한 뒤 순서 중복만큼 나눔(예: 2자리 부분집합 → $\dfrac{n(n-1)}{2}$).

요약: 문제 상황 → (조건화→시각화→적용 가능한 정리(AM-GM/C-S/판별식/삼각부등식 등) 선택) → 등호 조건·0 케이스 확인 → 결론

## 4. 증명 예시(짧게)

- 루트3 무리수 귀류법 요약:
  1. 가정: $\sqrt{3}=\dfrac{n}{m}$ (기약분수, $m,n\in\mathbb{N}$).
  2. 제곱: $3m^{2}=n^{2}$ → $n^{2}$ 는 3의 배수 → $n$ 은 3의 배수.
  3. $n=3k$ 대입: $3m^{2}=9k^{2}\Rightarrow m^{2}=3k^{2}$ → $m$ 도 3의 배수.
  4. $m,n$ 서로소라는 가정과 모순 → 원명제 참.

- 삼각부등식 제곱 증명 아이디어:
  - 양변 제곱(모두 $\ge0$)으로 전환 후 전개하여 비교 → $|a+b|^{2}\le(|a|+|b|)^{2}$ 증명.

## 5. 집합 연산 · 벤 다이어그램 해석 요령

- 합집합(U): “A 또는 B에 속함” → 그림에선 두 원 모두 칠하기.
- 교집합(∩): “A와 B에 동시에 속함” → 교차 영역만 칠하기.
- 여집합(Compl): 전체 영역에서 해당 집합 부분을 제외.
- 차집합 $A\setminus B$: A에서 B에 속하는 부분을 뺀 영역.
- 집합 표기 전환 연습:
  - $(A\cap B)^{c}=A^{c}\cup B^{c}$ 으로 시각화해서 영역 바꾸기.
  - $X\setminus Y = X \cap Y^{c}$ 로 항상 차집합을 교집합+여집합으로 표현 가능.
- 복잡한 표시(여러 영역 칠한 문제): 각 색칠 영역을 “A이고(B), C는 아니다” 식으로 문장화 → 수식화 → 간단화.

실전 팁: 문제에 Venn 그림이 없더라도 머릿속으로 각 영역의 조건(포함/미포함)을 먼저 말로 적어보면 전환 쉬움.

## 6. 조합·부분집합 계산 팁

- 기본: 각 원소는 포함 or 미포함 → 2가지 선택 → 전체 부분집합 $2^{n}$.
- “특정 원소는 반드시 포함/불포함” 이면 해당 자리는 고정(선택지 1), 나머지 원소만 고려.
- 크기 k의 부분집합 수: $\binom{n}{k}$ (순서 무시).
- 순서 고려 문제: $P(n,k)=\dfrac{n!}{(n-k)!}$, 집합 문제로 돌아갈 땐 나누기 $k!$ 등으로 보정.
- 원소 합/곱을 묻는 문제: 각 원소가 나타나는 횟수(또는 지수) 계산 후 합·곱으로 환산.

예: $A=\{1,2,3,4,5\}$ 에서 원소 수가 3인 부분집합들의 각 원소 합의 총합  
- 각 원소(예: 1)은 3개짜리 부분집합 중에서 몇 번 등장하는가? → $\binom{4}{2}=6$ 번.  
- 전체 합 = $(1+2+3+4+5)\times 6 = 15\times6=90$.

## 7. 반복되는 수학적 주제·패턴 분석 (Recurring themes)
- 보수성(Complementarity): 논리·집합 연산에서의 상보(부정) 구조가 반복. 드모르간이 핵심.
- 0 예외성: 곱셈·나눗셈에서 0이 모순 혹은 예외를 만들며 문제 함정으로 출제 빈번.
- 완전제곱·비음수성: $(\cdot)^{2}\ge0$가 여러 부등식 도출의 근원.
- 등호 조건의 중요성: AM-GM, C-S, 완전제곱 등에서 등호 성립 조건(동일성·비례 관계)이 최적화 문제의 핵심.
- 포함관계 ↔ 함의: 집합 포함($\subseteq$)과 논리적 함의($\Rightarrow$)의 1:1 대응으로 많은 명제를 집합 포함 문제로 환원 가능.
- 증명 전략의 재사용성: 증명법(대우·귀류·직접)은 다양한 명제에서 동일한 틀로 적용.

의미: 위 패턴을 의식하면 문제 유형 분류 및 적합한 해법(대우/귀류/AM-GM/C-S 등) 선택이 빨라짐.

## Key Takeaways (실전 적용형)
- 논리 명제를 집합 포함관계로 전환해 그림(벤)으로 직관 확인 → 증명/부정이 쉬워짐.
- 명제의 증명은 먼저 ‘대우(contrapositive)’가 더 쉬운지 검토. 가능하면 대우로 푼다.
- 부등식에서 곱셈·나눗셈 시 항상 분모·곱셈 인수의 부호(혹은 0 여부)를 명시. 시험의 함정 포인트.
- AM-GM → 최댓값·최솟값 문제 우선 후보. 등호 조건($a=b$)으로 해의 형태 즉시 판별 가능.
- 절댓값 관련문제: 삼각부등식(또는 제곱 전개)으로 전개하면 복잡도가 낮아짐.
- 집합 문제는 “문장(예: A이고 B이고 C가 아니다)”으로 각 영역을 정의한 뒤 연산식으로 바꿔라.
- 부분집합 수는 각 원소의 포함 여부 선택(2가지) → 조건부(고정 포함/불포함)시 남은 자리만 계산.

## 부족한 부분 / 개선 제안 (간결)
- 강의 전사에선 엄밀한 정의(명제 vs 조건)의 형식적 구분이 약함 → 시작부에 형식적 정의(논리기호·문법) 보강 권장.  
- 드모르간·대우의 증명(형식적 논리적 유도)은 직관 중심으로만 설명됨 → 간단한 진리표 보강 권장.  
- 코시-슈바르츠의 벡터·내적 관점 설명이 부족 → n차원 일반화(벡터) 예시 추가하면 응용력 향상.  
- 무리수 증명 예시는 √3 사용으로 충분하지만, 유한성·기약분수 조건 설명을 더 명확히 표기 권장.  
- 부분집합·조합 관련 공식(조합 표기와 순열의 차이)을 식으로 요약한 표 추가 권장.

## Insights (추가적 유용 지식 · 확장)
- De Morgan 일반화: 유한 또는 무한 인덱스 집합 $\{A_{i}\}_{i\in I}$ 에 대해  
  $(\bigcup_{i\in I} A_{i})^{c}=\bigcap_{i\in I} A_{i}^{c}$, $(\bigcap_{i\in I} A_{i})^{c}=\bigcup_{i\in I} A_{i}^{c}$.
- 코시-슈바르츠 일반화: 힐베르트 공간(내적공간)에서 $\langle u,u\rangle\langle v,v\rangle\ge|\langle u,v\rangle|^{2}$ — 벡터·직교성 문제에 직결.
- AM-GM 다항형 확장: 비음수 $a_{1},\dots,a_{n}$에 대해 $\dfrac{\sum a_{i}}{n}\ge(\prod a_{i})^{1/n}$, 등호 $a_{1}=\cdots=a_{n}$.
- 포함배제법 일반식(임의 n):  
  $|\cup_{i=1}^{n}A_{i}|=\sum_{k=1}^{n}(-1)^{k+1}\Big(\sum_{1\le i_{1}<\cdots<i_{k}\le n}|A_{i_{1}}\cap\cdots\cap A_{i_{k}}|\Big)$.
- 삼각부등식의 벡터·거리 해석: $|x+y|\le|x|+|y|$ 는 거리(유클리드)에서의 성질. 분석기하·벡터 문제에 자주 등장.
- 무리수 증명 계승: √p (p는 소수)는 무리수. 위 루트3 증명 구조를 일반 소수에 대해 확장 가능.

간단 예제(AM-GM 응용):
- 문제: $x+y=10$ 인 양수 $x,y$ 에 대해 $xy$의 최대값?  
  - AM-GM: $\dfrac{x+y}{2}\ge\sqrt{xy}\Rightarrow \sqrt{xy}\le5\Rightarrow xy\le25$. 등호 $x=y=5$.
