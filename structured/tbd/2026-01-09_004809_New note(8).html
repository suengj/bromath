핵심 수학(통계/확률) 개념 — 정의 → 성질 → 적용

목표: "특정 점포가 관찰된 당첨 횟수(예: k=3)를 보였을 때, 그 점포의 판매 비율(또는 연간 판매량)을 어떻게 추정하고 검정할지"에 대한 수학적 틀 제공.

1) 표기법(권장)
- W : 관찰 기간 전체 1등 당첨 티켓 총수(예: 연간 전체 1등 당첨 티켓 수, 토론에서는 52 사용)
- k : 특정 점포에서 관측된 1등 당첨 횟수 (관측값)
- p : 해당 점포의 티켓 점유율(전체 판매량 대비 비율) — 우리가 추정하려는 값
- n_draws : 관찰한 회차 수(예: 52주)
- q : 회차별(단일 회차) 해당 점포의 1등 당첨 확률(점유율과 티켓당 당첨 규칙에 따라 변환)

2) 기본 모델(가정: 당첨은 '티켓' 단위로 균등 랜덤)
- 가정A: 각 1등 당첨 티켓은 전체 판매 티켓 중 무작위로 선택 → 특정 점포가 차지하는 티켓 비율 p이면, W개의 당첨 중 특정 점포에서 k번 당첨될 확률은 이항분포
- K ~ Binomial(W, p)
- P(K = k) = C(W,k) p^k (1−p)^{W−k}
- 가정B: 각 회차에서 1등 당첨 수는 0/1(또는 회차별 유의미한 평균)로 모델링 가능 → 회차 단위 모델: 회차별 독립 확률 q (q ≈ p × E[1등 티켓 수 per draw / tickets per draw])

3) 추정(점추정, 신뢰구간)
- MLE(점추정): (\hat p = k / W)
- 근사 표준오차: (\mathrm{se}(\hat p) = \sqrt{\hat p(1-\hat p)/W})
- 정규근사 95% CI: (\hat p \pm z_{0.975}\,\mathrm{se}(\hat p)) (작은 W일 땐 Wilson 또는 exact CI 권장)
- 베이지안(균등 사전 α=β=1): posterior ( \mathrm{Beta}(k+1,\,W-k+1)). 사후 평균 ( (k+1)/(W+2)). 사후 예측 가능.

4) 가설검정(예: H0: p = p0 vs H1: p > p0)
- Binomial test(정확 검정) 사용: p-value = P_{H0}(K ≥ k) = (\sum_{i=k}^{W} {W \choose i} p_0^i (1-p_0)^{W-i})
- 근사 검정: z = ((\hat p - p_0))/sqrt(p_0(1-p_0)/W)
- 유의수준 α에 따라 귀무기각 여부 판단

5) 시간·회차 관점(대기시간 모델)
- 회차별 성공 확률 q이면, "다음 당첨까지의 회차"는 기하분포: E[waiting] = 1/q
- n회 내 "적어도 한 번" 확률: (1 - (1-q)^n)

6) 희소사건 근사
- W 크고 p 작을 때 K ≈ Poisson(λ = W p) 사용 가능 → 간단한 p 역추정과 p값 계산에 유용

7) 역문제(관찰 k로 p의 범위 추정)
- 관찰 k에 대해 p를 신뢰구간으로 역추정 → 해당 p 범위를 전체 판매량(예: S_total)와 곱해 점포 연간 판매량 추정

문제해결 구조(토론에서 제안한 절차)

  1. 데이터 수집
  2. W(기간 동안의 총 1등 당첨 티켓 수), 점포별 판매량(S_i), 전체 판매량 S_total (연 단위)
  3. 점포별 당첨 횟수 k_i (관찰값), 위치·피처(지하철 거리, 소득 수준 등)
  4. 단순 검정(빈도론)
  5. 가정 설정: H0(잠실 점포의 점유율 p0 = S_expected/S_total) vs 관찰 k
  6. Binomial test 또는 정규근사(또는 exact)
  7. 추정 / 신뢰구간
  8. (\hat p = k/W), CI 계산(Wilson 또는 Clopper-Pearson 권장)
  9. 모델 확장
  10. 회귀/패널: 점포 피처 X → 로그 판매량 또는 점유율 예측
  11. 베이지안 계층모델: 점포별 p_i ~ Beta(α,β), 관측 k_i | p_i ~ Binomial(W, p_i)
  12. 추가 고려사항
  13. 다중비교(여러 점포 동시 검사 시 FDR 보정), 시간에 따른 변화(패널/생존분석), 공간상관(인접 점포 영향), 복수 당첨(한 회차 다수 당첨 가능성)

주요 수식 요약(참고)

  • 이항확률: (P(K=k)=\binom{W}{k} p^k (1-p)^{W-k})
  • MLE: (\hat p = k/W)
  • SE: (\sqrt{\hat p(1-\hat p)/W})
  • at-least-one (n회): (1-(1-q)^n)
  • 기댓값(기하): (E[T]=1/q)
  • 베이지안 posterior: (p|k \sim \mathrm{Beta}(k+\alpha, W-k+\beta))

Key Takeaways (실무적·행동 가능한 항목)

  • 관찰치 k와 전체 당첨수 W가 주어지면 우선점: 점추정 (\hat p=k/W) 계산. (간단·직관적)
  • 신뢰구간은 Wilson 또는 Clopper–Pearson 방식으로 구해 불확실성 명시. 정규근사만 쓰지 말 것(특히 W 작거나 p 극단치).
  • 가설검정(H0: p = p0): exact Binomial test로 p-value 산출. 여러 점포를 검사하면 다중비교 보정(FDR 등) 적용.
  • 만약 p를 전체 판매량으로 환산하려면: S_store ≈ p × S_total (S_total = 분석 기간의 전체 판매량)
  • 회차 기반 예측: 회차별 확률 q를 이용해 다음 당첨까지 기대 대기회차 E[T] = 1/q 계산.
  • 베이지안 접근 권장: Beta 사전 → Beta posterior 사용하면 불확실성 정리·사전 정보 반영 쉬움.
  • 데이터 요구사항(우선 확보): 점포별 판매량(연/회차별), 점포별 당첨 건수, 기간별 전체 판매량, 점포 메타데이터(위치·영업시간·인근시설·지역소득 등).
  • 모델 확장(권장 실행 순서):
  • 표준화된 이항검정(단일 점포)
  • 여러 점포 비교 → 다중비교 보정
  • 회귀(로그it 또는 비율 회귀)로 피처 포함
  • 베이지안 계층모델로 점포 간 변이·불확실성 포착
  • 시공간 모델(패널/공간상관)로 시간·이웃영향 반영

  12. 제안: 다중비교 보정, FDR, 또는 베이지안 사전으로 보정.

Insights (추가 수학적·실무적 지식, 계산 예시 포함)
- 베이지안 간단 예시(관찰 k=3, W=52, 균등사전 α=β=1)
- Posterior: (p|k \sim \mathrm{Beta}(4, 50))
- 사후 평균: (E[p|k]=(k+1)/(W+2)=4/54\approx0.07407)
- 95% credible interval: Beta(4,50)의 2.5%/97.5% 분위수 이용(수치계산 필요)
- 해석: 표본이 약소하므로 베이지안으로 불확실성 직관적 표현 가능
- 빈도론 예시(정규근사)
- (\hat p=3/52\approx0.0577)
- se ≈ sqrt(0.0577×0.9423/52) ≈ 0.0323
- 95% CI ≈ 0.0577 ± 1.96×0.0323 ≈ [0, 0.121] (하한 0으로 절단)
- 해석: p가 0.02인지 0.10인지 확정하기엔 표본 불충분
- 포아송 근사
- W 큰데 p 작을 때 K ~ Poisson(λ=W p). 관찰 k로 λ 추정: (\hat\lambda=k).
- 예: k=3 → λ≈3 → p≈3/W
- 역추정(판매량 환산)
- 전체 연간 판매량 (S_{total}) 알고 있을 때 점포 연간 판매량 추정: (S_{store}\approx \hat p \times S_{total})
- 예: S_total = 30,000장(연), (\hat p \approx 0.0577) → S_store ≈ 0.0577×30,000 ≈ 1731장/연
- 여러 점포 비교(다중검정)
- 여러 점포에서 k_i 관찰 → 각 점포 p_i 추정 → 많은 점포 중 상위 몇 곳이 '유의'하면 우연 가능성 높음(look-elsewhere). FDR 절차(Benjamini–Hochberg) 권장.
- 공간/패널 모델 제안(수식적 아이디어)
- 로그-비율 회귀: (\log\left(\frac{p_i}{1-p_i}\right) = X_i\beta + u_i) (X_i = 피처들, u_i = 점포 고유효과)
- 베이지안 계층: (p_i\sim\mathrm{Beta}(\alpha,\beta)), (k_i|p_i\sim\mathrm{Binomial}(W,p_i))
- 관찰자 편향(인지적)
- 사후관찰→행동(구매)→관찰증폭: 통계적 귀속(귀인) 오류. 모델링시 '사후확인 효과' 변수로 통제 권장.
- 추가 모델링 팁
- 작은 W: Wilson CI 또는 exact(Clopper–Pearson)
- 희귀 이벤트: Poisson 또는 zero-inflated 모델 고려
- 여러 해 데이터: 패널로 전환 → 고정효과/시간효과 추정
- 모형 비교: AIC/BIC, 베이지안 경우 WAIC/LOO-CV 사용

실전 계산(권장 워크플로우)
1. 수집: W, {k_i}, S_total, {S_i if available}, 점포 피처
2. 단순 분석: 각 점포 (\hat p=k/W), Wilson CI
3. 가설검정: Binomial exact test (H0 예: p equals baseline)
