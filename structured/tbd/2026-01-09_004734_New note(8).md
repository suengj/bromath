## 2) 핵심 수학·통계 개념 (핵심만)
- 표본·모수:
  - N = 연간 전체 판매 티켓 수(전국)  
  - J = 특정 점포(명당)의 연간 판매 티켓 수(모수, 추정 대상)  
  - n = 관측 횟수(연간 추첨 횟) — 통상 n = 52 (주 단위)  
  - K = 특정 기간(예: 1년) 동안 해당 점포에서 나오는 1등 수(관측값, 정수)

- 확률모형(단순 가정):
  - 각 회차에서 1등 당첨 티켓이 그 점포에서 나올 확률 p = J / N (각 티켓이 동일 확률로 당첨된다는 가정).
  - K ~ Binomial(n, p) = Bin(n, J/N).
  - 우도: L(p) ∝ p^K (1−p)^{n−K}.

- 추정:
  - MLE: \(\hat p = K/n\). 따라서 \(\hat J = \hat p \cdot N = (K/n)N\).
  - 신뢰구간: 이항 분포 기반(예: Clopper–Pearson), 근사로 정규 근사 혹은 포아송 근사(λ = n p) 사용 가능.

- 가설검정(예):
  - 귀무가설 H0: p = p0 (또는 J = J0) vs 대립가설 H1: p > p0 등.
  - p-value = P_{H0}(K ≥ observed K) = \(\sum_{i=K}^{n} \binom{n}{i} p_0^{i}(1-p_0)^{n-i}\).

- 베이즈 대안:
  - 사전: p ~ Beta(α, β) → 사후: p | data ~ Beta(α+K, β+n−K).
  - 사후 신뢰구간(credible interval)으로 J 분포 유도 가능: J = p·N.

- 주의할 통계적 오류:
  - 사후 선택(post-selection bias): “명당”이라 불린 장소는 이미 관심 대상으로 선택된 곳 → 다중비교·선택편향 보정 필요.
  - 비정상성: 시간에 따라 J가 변함(명성 효과로 판매 증가) → 정적 모델 부적합.
  - 관측의 독립성 가정 위반: 한 회차에 여러 1등(공동당첨자) 발생 등.

## 3) 원문에서 논의된 계산·예시(정리된 수식과 숫자)
원문 토의 핵심 예시: 연간 전체 티켓 N = 30,000, n = 52회, 관측 K = 3.

- MLE:
  - \(\hat p = 3/52 \approx 0.0577\).
  - \(\hat J = 0.0577 \times 30000 \approx 1{,}731\) 장/년.

- 포아송 근사(λ = n p0)로 P(K ≥ 3) 계산(근사):
  - 만약 p0 = 600/30000 = 0.02 → λ = 52×0.02 = 1.04  
    P(K ≥ 3) ≈ 1 − e^{-λ}(1 + λ + λ^2/2) ≈ 0.089 (약 8.9%).
  - p0 = 300/30000 = 0.01 → λ = 0.52 → P(K ≥ 3) ≈ 0.016 (1.6%).
  - p0 = 3000/30000 = 0.10 → λ = 5.2 → P(K ≥ 3) ≈ 매우 큼(대부분의 경우 3 이상).

- 역문제(유의 수준 α = 0.05 에 해당하는 J):
  - λ ≈ 0.8 일 때 P(K ≥ 3) ≈ 0.05 → p ≈ 0.8/52 ≈ 0.0154 → J ≈ 0.0154×30000 ≈ 462.
  - 해석: N=30000 가정하에, 한 점포가 연간 ≈462장(전체의 ≈1.5%)만 팔아도 1년 동안 3회 이상 당첨될 확률이 ≈5%로 ‘드문 사건’으로 보기 힘듦.

- 결론(숫자 정리):
  - 관측 K=3은 J가 600 이상이면(약 p≥0.02) 흔하지 않은 사건은 아님(약 9% 수준).
  - 관측만으로 “조작” 또는 비정상적 현상으로 결론 내리려면 더 강한 증거(더 많은 관측·시간범위·비교군)를 필요.

## 4) 실무적 모델링·검정 절차 제안 (액션 블루프린트)
1. 데이터 수집(우선순위)
   - 전국 총 티켓 판매량 N(주 단위·연 단위).  
   - 각 점포별 주(회)별 판매량 J_{i,t}.  
   - 각 회차별 1등 티켓의 판매 점포 정보(있다면 일자·티켓ID).  
   - 점포 메타데이터: 위치(지하철 접근성), 상권 특성, 인구·소득 지표, 명성(언론·SNS 언급) 등.

2. 탐색적 분석
   - 각 점포별 연·주별 판매 비율 p_{i,t} = J_{i,t}/N_t 분포 확인.  
   - 후보 명당(사후 선택된 점포)과 일반 점포의 p 분포 비교(박스플롯, 히스토그램).

3. 단순 검정(초기)
   - 각 후보 점포에 대해 H0: p = p̄ (전체 평균) → 이항검정 또는 정규 근사.  
   - 다중 비교 보정(예: Bonferroni, BH) 적용.

4. 계층모델(권장)
   - 베타-이항 계층모형: p_i ~ Beta(α,β), K_i | p_i ~ Bin(n, p_i).  
   - 장점: 개별 점포의 사후 추정치가 풀(pooling)되어 극단치 완화(shrinkage).  
   - 확장: 시간 차원 추가(패널모형), covariate 포함(p_i(t) = logistic(X_{i,t}β + u_i + ε_{i,t})).

5. 베이지안 접근(권장)
   - 사전 정보(예: 전체 판매량 분포) 반영.  
   - 사후 신뢰구간 제공으로 불확실성 투명화.  
   - 시뮬레이션(모의실험)으로 다양한 J 시나리오에서 관측될 K 분포 확인.

6. 민감도 분석
   - N, n, 공동당첨(여러명 당첨시 처리), 당첨 티켓이 여러 장일 경우 가중치 등 가정 변경에 따른 민감도 확인.

7. 보고
   - p̂, Ĵ, 신뢰구간/credible interval, p-value, 다중비교 보정 결과, 모델 기반 추정치 및 예측(향후 k의 분포).

## Key Takeaways (실행 가능한 핵심)
- 관측 k(예: 연간 3회)는 단순 관측만으로는 “조작” 결론에 충분치 않다. J(판매량) 추정·CI 확인 필수.  
- 최소 데이터: 각 회차 당 전국 판매량 N_t, 점포별 J_{i,t}, 각 회차 1등 발생 위치.  
- 권장 방법: 계층 베타-이항 모델 또는 베이지안 접근으로 점포별 p 추정(풀링으로 안정화).  
- 다중 후보(여러 명당) 검정 시 다중비교 보정 필수(Bonferroni/False Discovery Rate).  
- 사후 관측을 사전 신념으로 오해하지 않도록, 사전확률(유의수준)과 사후확률(관측 후 확률)을 명확히 구분.

## Insights (추가적 수학·통계적 권장사항)
- MLE의 직관: 관측 K=3, n=52이면 p̂=0.0577. 그러나 표본크기 작음 → 신뢰구간 넓음. Clopper–Pearson 95% CI로 p의 하한을 확인하면 “진짜 많이 팔리는 지” 여부 판단에 유용.  
- 베이즈 해석 장점: p 분포의 불확실성을 직접 제공. 예: 사전 Beta(1,1) (균등) 사용 시 사후 Beta(4,50) → 사후 평균 ≈ 4/54 ≈ 0.074.  
- 계층모형 예시(수식):
  - p_i ~ Beta(α, β)  
  - K_i ~ Binomial(n, p_i)  
  - α,β는 전체 데이터로부터 추정(또는 약한 사전 지정).  
  - 효과: 개별 점포 추정치는 전체 분포로 수축(shrinkage), 극단치 과대평가 방지.  
- 시뮬레이션 권장: 실제 N, J 분포(관찰치 기반)로 Monte Carlo 시뮬레이션 해, “3회 이상” 사건의 빈도 분포를 얻어 실제 관측의 희귀성 판단.  
- 다중검정과 선택편향: 표본 내에서 “가장 많이 당첨된” 점포만 따로 분석하면 극단선택 문제 발생. 해결: 전체 점포군에서 각 점포의 p분포를 동시에 모델링.
