정리(수학적 내용과 흐름)
1) 기본 정의(핵심)
- 지수: a의 자연수 거듭제곱 a^n = a·a·...·a (n번).
- 로그: log_a b = c ⇔ a^c = b (a>0, a≠1, b>0).
- 로그와 지수는 서로의 역함수: a^{\log_a x} = x, \log_a(a^x) = x.

2) 기본 성질(핵심 수식)
- 지수법칙:
  - a^{m}·a^{n} = a^{m+n}
  - a^{m}/a^{n} = a^{m-n}
  - (a^{m})^{n} = a^{mn}
  - (ab)^{n} = a^{n} b^{n}
- 로그 성질:
  - \log_a(xy) = \log_a x + \log_a y
  - \log_a(x/y) = \log_a x - \log_a y
  - \log_a(x^{r}) = r \log_a x
  - Change of base: \log_b x = \dfrac{\log_a x}{\log_a b}

Insights (추가 수학적 보완·심화 — 교육용 및 실무적 관점)
1) 지수와 로그의 확장(요약)
- 자연수 지수: a^n = a·a·...·a (n회).  
- 정수 지수: a^{0}=1, a^{-n} = 1/a^{n}. (지수법칙을 보존하려는 정의)  
- 유리수 지수: a^{m/n} 정의는 a^{m/n} := (a^{1/n})^{m} (단, a>0, n∈N).  
- 실수 지수: 보통 자연로그와 지수함수(exp)를 이용해 정의: a^{x} := e^{x\ln a} (a>0). 이 정의는 연속성과 미분 가능성을 보장.

2) 로그의 기초적·응용적 성질(몇 가지 강조)
- 로그는 지수함수의 역함수 → 단조성:
  - a>1이면 \log_a x는 증가함수.
  - 0<a<1이면 \log_a x는 감소함수.
- 치역·정의역: \log_a x의 정의역은 x>0, 치역은 모든 실수.
- 역함수 성질 응용: 방정식 a^{f(x)} = a^{g(x)} ⇒ f(x)=g(x) (a>0, a≠1).
- Change-of-base 공식 실전 팁: 계산기 로그는 주로 ln(자연로그)나 log_{10}이므로 \log_a b = \dfrac{\ln b}{\ln a}를 기억하면 실전 편리.

3) 미적분과의 연결(간단 정리)
- 미분:
  - \dfrac{d}{dx} a^{x} = a^{x} \ln a
  - \dfrac{d}{dx} \ln x = \dfrac{1}{x}
- 적분 관련: \int a^{x} dx = \dfrac{a^{x}}{\ln a} + C, \int \dfrac{1}{x} dx = \ln|x| + C
- 응용: 성장/감소 모델, 반감기, 지수적 성장(금융·인구·방사성 붕괴 모델 등)

4) 실전 예제(대표형 · 간결 풀이)
예제 A — 계산형(전형)
문제: \left(\dfrac{a^{2}}{a^{\sqrt{2}}}\right)^{\sqrt{2}} 을 간단히 하라. (a>0)
풀이: 내부 지수 차: a^{2-\sqrt{2}}. 전체 거듭제곱: (a^{2-\sqrt{2}})^{\sqrt{2}} = a^{(2-\sqrt{2})\sqrt{2}} = a^{2\sqrt{2} - 2}.
핵심: (a^{m}/a^{n})^{p} = a^{(m-n)p}.

예제 B — 방정식형
문제: \log_2(3x+1) - \log_2(x-1) = 2. (정의역 고려)
풀이:
1) 로그 차 → 로그 곱/나눗셈 성질: \log_2\left(\dfrac{3x+1}{x-1}\right)=2.
2) 지수형으로 변환: \dfrac{3x+1}{x-1} = 2^{2}=4.
3) 선형 방정식: 3x+1 = 4x - 4 ⇒ x = 5. 정의역 확인: x>1 → x=5 유효.

5) 문제 문해력(구체적 연습법)
- 연습 1: 긴 조건문을 받으면 “묻는 것 한 문장으로 요약”하기. 예: "t에 대해 x의 서로 다른 실근의 개수" → 목표: g(t)=서로 다른 실근의 수를 구하는 문제.

Insights (추가 자료: 심화·참고)
- 자연지수 e와 지수함수:
  - e := \lim_{n\to\infty}\left(1+\dfrac{1}{n}\right)^{n}.  
  - a^{x} = e^{x\ln a}를 이용하면 실수승의 성질·미분이 자연스럽게 다뤄짐.
- 로그 단위 변환 실무 팁:
  - 계산기에서 \log_{10}, \ln만 지원할 때는 change-of-base 이용: \log_a b = \dfrac{\ln b}{\ln a}.
- 로그 방정식에서 유효해지는 변환(경계조건 주의):
  - 로그 취하기 전/후 정의역 체크 필수(로그 내부 >0).  
  - 지수 취한 뒤에는 역함수 특성 때문에 extraneous root(거짓해)는 보통 발생하지 않지만, 취급 중 변형(양변을 전부 제곱 등 비역함수적인 조작) 시 주의.
- 그래프적 판단(근의 개수 토론에 유용)
  - f(x)=a^{g(x)}와 h(x)와의 교점문제 → g(x)의 단조성, 연속성 파악 후 비교.  
  - 로그 식의 교점 수 판단은 정의역(x>0)과 단조성으로 빠르게 제한할 수 있음.
- 추천 연습문제(실전 감각 향상)
  1. \log_3(x+1) + \log_3(x-2) = 2. (정의역 확인 포함)  
  2. Solve: 2^{x} + 2^{x+1} = 3^{x+2}. (양변 로그 취하고 선형화로 접근)  
  3. 주어진 함수 f(x)=\log_{2}(x^{2}-3x+2) 의 정의역과 그래프 특징 분석.
