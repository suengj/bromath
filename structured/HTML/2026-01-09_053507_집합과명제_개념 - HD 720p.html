핵심 개념 (Core Concepts)

(간결히 정의·정리 — 각 항목은 바로 사용할 수 있는 명확한 진술)

  1. 명제와 조건
  2. 명제: 참/거짓이 명백히 결정되는 문장(예: "3은 소수이다").
  3. 조건(조건문): 변수값에 따라 참/거짓이 달라지는 문장(예: "$x$는 짝수이다").
  4. 부정: 조건 $P$의 부정은 $\neg P$.

  5. 진리값 규칙: 명제 $Q$가 참이면 $\neg Q$는 거짓, 거짓이면 $\neg Q$는 참.

  6. 논리 연결과 포함관계

  7. 함의(implication): "$P$이면 $Q$이다" ($P\Rightarrow Q$). 의미: $P$가 참이면 $Q$도 항상 참.
  8. 역 (converse): $Q\Rightarrow P$.
  9. 대우 (contrapositive): $\neg Q \Rightarrow \neg P$ — 함의의 대우는 항상 원래 명제와 동일한 진리값을 가짐.

  10. 필요조건 / 충분조건:

    • $P\Rightarrow Q$이면 "$P$는 $Q$의 충분조건", "$Q$는 $P$의 필요조건".
    • 쌍방($P\Leftrightarrow Q$): 필요충분조건(동치).

  11. 집합과 연산

  12. 원소 표기: $a\in A$, $b\notin A$.
  13. 부분집합: $C\subseteq B \Leftrightarrow (\forall x)(x\in C \Rightarrow x\in B)$.
  14. 집합 연산:
    • 합집합: $A\cup B$ (또는 또는).
    • 교집합: $A\cap B$ (동시).
    • 차집합: $A\setminus B$ (A에 속하지만 B에는 속하지 않는 원소들의 집합).
    • 여집합: $A^c$ (전체 집합에서 $A$를 뺀 것).

  15. 드모르간 법칙 (De Morgan):

    • $(A\cap B)^c = A^c\cup B^c$
    • $(A\cup B)^c = A^c\cap B^c$

  16. 진리집합

  17. 조건 $p$의 진리집합 $\mathrm{P}$: 전체집합 안에서 $p$를 참으로 만드는 원소들의 집합.

  18. 함의 $P\Rightarrow Q$는 집합 포함관계 $\mathrm{P}\subseteq\mathrm{Q}$와 동치.

  19. 부등식(기본 규칙)

  20. 수직선의 관점: 오른쪽이 크다. $A>B \iff A-B>0$.
  21. 덧셈/뺄셈: 모든 실수 $c$에 대해 $A>B \iff A+c > B+c$.
  22. 곱셈: $A>B$이면
    • $c>0$이면 $Ac > Bc$,
    • $c<0$이면 부등호 방향이 바뀜: $Ac < Bc$.

  23. 나눗셈은 곱셈의 역수 관점으로 동일 규칙(분모는 0 아님 주의).

  24. 절대(부등)식 (항등적 부등식)

  25. 절대부등식: 모든 실수에 대해 성립하는 부등식(예: $x^2\ge0$).

  26. 절대값 관련:

    • 삼각부등식: $|a+b| \le |a|+|b|$ (등호 조건: $a,b$의 부호가 같을 때).
    • $|x|=0 \iff x=0$; $x^2+y^2=0 \iff x=y=0$.

  27. 이차식(discriminant)과 그래프 관점

  28. 이차함수 $f(x)=ax^2+bx+c$의 판별식 $\Delta=b^2-4ac$:
    • $\Delta<0$이면 실근 없음 → $f(x)$는 $a>0$일 때 항상 양수(위로 볼록) 등.

  29. 판별식을 이용해 부등식의 항등성(항상 양수 등)을 파악.

  30. 주요 부등식들

  31. 산술–기하 평균(AM-GM) (두 항): $\displaystyle \frac{a+b}{2}\ge\sqrt{ab}$ ($a,b\ge0$). 등호 조건: $a=b$.
  32. 코시–슈바르츠 (Cauchy–Schwarz) (두 벡터):
    • $(a^2+b^2)(x^2+y^2) \ge (ax+by)^2$,
    • 일반화: $\sum_i a_i^2 \cdot \sum_i x_i^2 \ge \left(\sum_i a_ix_i\right)^2$.

  33. 완전제곱식: $(a\pm b)^2 \ge 0$ 등을 활용한 부등식 증명.

  34. 조합(combinatorics)

  35. 부분집합 개수: 유한집합 $|A|=n$이면 모든 부분집합의 수는 $2^n$.
  36. 크기 $k$인 부분집합 수: $\binom{n}{k}$.
  37. 포함배제(두 집합): $|A\cup B| = |A| + |B| - |A\cap B|$.

  38. 세 집합 확장: $|A\cup B\cup C| = \sum|A_i| - \sum|A_i\cap A_j| + |A\cap B\cap C|$.

  39. 증명 기법 (요약)

    • 직접증명: 가정에서 출발해 결론 도출.
    • 대우증명: $P\Rightarrow Q$ 를 $\neg Q \Rightarrow \neg P$로 증명.
    • 귀류법(모순법): 결론의 부정을 가정해 모순을 도출 → 원명제 참.
    • 판별식·완전제곱·부등식 이용: 함수적 관점 또는 대수 변형으로 증명.

성질 / 공식 표 (한눈 정리)

1) 논리 ↔ 집합 대응

논리 표현 집합 대응
$P\Rightarrow Q$ $\mathrm P \subseteq \mathrm Q$
$\neg(P\wedge Q)$ $\neg P \vee \neg Q$ (드모르간)
$\neg(P\vee Q)$ $\neg P \wedge \neg Q$ (드모르간)
$P\vee Q$ $P\cup Q$
$P\wedge Q$ $P\cap Q$

2) 부등식 요약

  • 덧셈/뺄셈: 항상 유지.
  • 곱셈: 부호에 따라 유지/역전.
  • 나눗셈: 분모·분자 부호 주의.
  • 양변 제곱: 양변이 비음수일 때 비교 보존(특히 절대값).

3) 주요 부등식: (등호 조건 포함)

  • AM-GM: $\frac{a+b}{2}\ge\sqrt{ab}$ ($a,b\ge0$, 등호 iff $a=b$).
  • Cauchy: $(\sum a_i^2)(\sum x_i^2)\ge(\sum a_ix_i)^2$ (등호 iff $a_i$와 $x_i$는 비례).
  • 삼각부등식: $|a+b|\le|a|+|b|$ (등호 iff $a,b$ 동호 부호).

문제 해결 구조 (Problem-Solving Structure)

(문제문 → 접근·전개 → 핵심기술 → 결론)

  1. 문제 읽기: 명제가 참/조건문인지 판별 → 변수를 고정(전체집합) → 진리집합 표기 추천.
  2. 전개 전략 선택:
  3. 포함 관계 판단 문제 → 진리집합 표기 후 포함성 증명.
  4. 부등식 문제 → 부호 경우 분류 또는 제곱/완전제곱으로 환원.
  5. 최대·최소 문제 → AM-GM, Cauchy, 완전제곱 또는 미분(시험분야 성격에 따라).
  6. 조합 문제 → 자리별 선택/순열과 조합(순서 무시) 판단, 혹은 이항계수 $\binom{n}{k}$ 사용.
  7. 증명 기법:
  8. 적당한 가정(예: $a,b\ge0$)을 확인하고 필요 시 경우 나눔.
  9. 등호 조건은 증명 과정에서 만드는 제곱항(=0)으로 읽음.
  10. 검증: 경계조건(분모 0, 음수 곱셈)과 등호 가능성 확인.

선택된 상세 설명 · 예제

  1. 드모르간 법칙(집합)
  2. 그림(벤다이어그램)으로 직관 확인 가능.

  3. 예: $(A\cap B)^c$는 $A$와 $B$의 교집합 바깥 전체 → $A^c\cup B^c$로 표현.

  4. 부등식 곱셈 주의 예

  5. 명확히: $A>B$이고 $C$가 미지수일 때, $C$의 부호를 모르면 $AC$와 $BC$ 비교 불가.

  6. 시험상의 함정: 문제에서 "$AC\ne BC$"가 항상 참인가? (거짓: $C=0$ 예시).

  7. 판별식을 이용한 절대부등식 예

  8. $x^2+x+1$의 판별식 $\Delta=1-4<0$ → 모든 실수에 대해 양수.

  9. 즉 $x^2+x+1>0$ (항상, 실수 범위).

  10. AM-GM 사용 예 (최소값 문제)

  11. 문제: $a,b>0$이고 $a+b$ 고정일 때 $ab$의 최대는?
  12. 해법: AM-GM로 $ab\le\left(\frac{a+b}{2}\right)^2$ → 등호는 $a=b$.

  13. 응용: 최대·최소 문제에서 흔히 사용.

  14. Cauchy 증명 스케치 (2차원)

  15. 전개: $(a^2+b^2)(x^2+y^2) - (ax+by)^2 = (ay-bx)^2 \ge 0$ → 성립, 등호 조건: $ay=bx$ 즉 벡터 비례.

  16. 귀류법 예 — $\sqrt{3}$의 무리수성

  17. 가정: $\sqrt3 = \frac{m}{n}$ (서로소, 자연수).

  18. 유도: $3m^2=n^2$ → $n$는 3의 배수 → $m$도 3의 배수 → 서로소 모순 → 무리수.

  19. 부분집합 개수와 합/곱 문제

  20. $A={1,2,3,4,5}$에서 크기 3인 부분집합 개수: $\binom{5}{3}=10$.
  21. 크기 3 부분집합들의 원소 총합(예시): 각 원소가 등장하는 횟수를 계산(대칭성) → 각 원소는 $\binom{4}{2}=6$번 등장 → 총합 $6(1+2+3+4+5)=90$.

  22. 곱의 총곱: 각 원소가 포함되는 경우의 수만큼 지수승 곱셈.

  23. 포함배제(원소수 직관)

  24. 두 집합: $|A\cup B| = |A| + |B| - |A\cap B|$ (중복제거).
  25. 세 집합으로 확장 시 + - + 꼴의 규칙을 기억.

반복되는 주제(Themes) 및 의미 분석

  1. 집합 ↔ 명제(진리집합) 대응
  2. 명제 논리의 산술적/집합적 해석은 문제 형태 전환에 핵심.

  3. 포함관계를 이용하면 많은 문제(함의, 대우, 여집합)를 시각적으로 처리 가능.

  4. 등호 조건은 항상 “완전제곱 → 0” 또는 비례(벡터) 형태로 귀결

  5. AM-GM, Cauchy, 삼각부등식 등에서 등호는 특정 구조(동일·비례)를 요구.

  6. 문제 풀이 시 등호 조건을 항상 확인해 해의 성질 파악.

  7. 부등식에서의 0(영)의 취급이 중요

  8. 곱셈/나눗셈에서 $0$은 특수 케이스(역원 없음).

  9. 시험 문제에서 $0$을 빠트리는 장난(예: $AC=BC$ 가능성) 빈번.

  10. 대수적 변형 ⇔ 함수적(그래프) 직관의 병행 사용

  11. 판별식으로 이차함수의 전 범위 값을 파악하면 부등식 판단이 쉬움.
  12. 대수적 조작(완전제곱)과 그래프적 직관을 함께 이용.

자주 발생하는 실수·주의점(간단 목록)

  • 분모가 0인 경우를 고려하지 않음.
  • 곱셈 시 부호(음수)에 의한 부등호 방향 전환 실수.
  • 명제의 부정에서 "모든"과 "어떤"의 변환 규칙을 혼동(부정: $\forall\rightarrow\exists\ \neg$, $\exists\rightarrow\forall\ \neg$).
  • 집합 문제에서 진리집합을 일부만 나열하는 실수(완전성 부족).
  • 등호 조건을 증명 과정에서 체크하지 않아 해의 성질이 틀리는 경우.

추가 연구·보충 인사이트(분리 섹션)
(아래는 원문 내용을 보완하는 외부 지식·실무적 팁·확장 사항)

Key Takeaways (실전 적용 가능한 요점)

  • 논리 명제와 집합의 1:1 대응을 익혀라: 함의 = 포함관계로 문제를 전환하면 직관적으로 해결 가능.
  • 부등식 문제에서 제일 먼저 할 것: 0 관련 특수 케이스(분모·곱셈) 체크, 등호 조건 확인.
  • 최대/최소 문제 우선순위: AM-GM → Cauchy → 완전제곱 → 미분(고급). 가능하면 먼저 AM-GM 적용 시도.
  • 조합 문제는 "자리를 선택하는 관점"과 "순서 무시"를 명확히 구분하여 계산(순열→조합 변환 시 나눗셈).
  • 증명의 전략 선택: 직접증명 vs 대우 vs 귀류법 — 함의 형태이면 대우가 유용, 무리수 증명 등에는 귀류법이 강력.

Insights (심화·확장 · 참고)

  • Cauchy–Schwarz는 벡터 내적 관점: $\langle \mathbf a, \mathbf x\rangle^2 \le |\mathbf a|^2|\mathbf x|^2$ — 고등학습: Hölder 불평등, Minkowski 부등식으로 확장.
  • AM-GM 일반화: $ \frac{a_1+\cdots+a_n}{n}\ge (a_1\cdots a_n)^{1/n}$ ($a_i\ge0$) — 등호 조건: 모두 같음.
  • 판별식 해석을 더 정교하게: $ax^2+bx+c$의 부호 결정은 $a$와 $\Delta$ 조합으로 완전하게 분류 가능(실근의 존재·위치 판단).
  • 집합 연산을 기하(벤다이어그램)→대수(집합식)→카운팅(원소수)로 전환하는 훈련이 문제 풀이 속도 향상에 유리.
  • 논리와 집합의 형식화: 양화사(∀, ∃) 엄격 사용을 권장(시험에서 부정·논리 변형에서 실수 줄임).

개선이 필요한 영역(간결한 지점·권장 보완)

  • 복소수에서의 "그래프" 언급은 불분명: 실수 범위를 전제로 한 함수 그래프 해석과 복소평면의 그래프(실수함수와 다름)를 구분하여 설명 권장.
  • 일부 설명에서 수식 표기·변수 조건($a,b>0$ 등)을 명시적으로 적지 않아 오해 발생 가능 — 문제마다 전제조건을 첫 줄에 명시하도록 권장.
  • 집합 표기와 명제 표기(소문자 조건 vs 대문자 진리집합)를 처음부터 엄격히 분리 설명하면 초급 수험생 혼동 감소.
  • 조합 관련: 순열·조합의 유도(자리선택 → 나누기 팩토리얼) 원리 간단한 표로 보완 권장.

  • 벤 다이어그램에서 "A이면서 B이지만 C는 아님" → $A\cap B \setminus C$ 혹은 $(A\cap B)\cap C^c$.

간결 정리 마무리
- 핵심은 '정의(명제·집합) → 집합 연산(직관·드모르간 등) → 부등식의 기본 규칙(특히 0·음수 처리) → 표준 부등식(AM-GM, Cauchy) → 조합 원리(자리선택/순서 비고)'의 흐름을 체계적으로 숙지하는 것.
- 실전 팁: 문제를 보면 먼저 전제(변수 범위, 0 여부, 음수 가능성)를 체크하고, 집합문제는 진리집합 표기, 부등식은 0 관련 특수 케이스부터 처리.
