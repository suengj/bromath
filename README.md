# BroMath - Transcribed and Structured Content

이 저장소는 BroMath 프로젝트의 전사(transcribed) 및 구조화(structured)된 콘텐츠를 포함합니다.

## 폴더 구조

- `transcribed/`: 오디오/비디오 파일에서 전사된 원본 텍스트 파일 (.txt)
- `structured/`: GPT를 통해 구조화된 마크다운 문서 (.md, .html)

## 내용

수학 교육 콘텐츠(강의, 토론, 리뷰)를 자동으로 전사하고 구조화한 문서들입니다.
